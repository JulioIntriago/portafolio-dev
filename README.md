# Portfolio responsive

Una web limpia y agradable para diseñador o programador. Eso incluye casi todo lo que quieres mostrar en tu primer portafolio. Al principio, tu nombre, enlaces a redes sociales y una gran imagen. Luego, la sección Acerca de muestra más información sobre usted. Luego la lista de Habilidades que ofrece. También tiene una sección donde muestra algunos trabajos recientes. Y en la parte inferior tiene un formulario, y al final con un pie de página.
